# birdnet-website
BirdNET website built with Eleventy.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/birdnet-team/birdnet-website.git
   cd birdnet-website
   ```

2. Build and run the site locally:
   ```bash
   npm install
   npm run serve
   ```
